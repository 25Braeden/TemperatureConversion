# Temperature Conversion Program

This Java program allows you to perform temperature conversions and generate conversion tables. You can convert between Fahrenheit, Celsius, and Kelvin temperatures, as well as generate conversion tables for a specified range of values.

## Features

- Convert temperatures between Fahrenheit, Celsius, and Kelvin.
- Generate conversion tables for Fahrenheit, Celsius, and Kelvin.
- Choose the temperature units for conversion and table generation.

## Usage

1. **Compile the Program:**

   Open your terminal or command prompt, navigate to the directory containing the `TempConversion.java` file, and compile it using the Java compiler:

   ```sh
   javac TempConversion.java
   ```

2. **Run the Program:**

   Run the compiled Java program using the following command:

   ```sh
   java TempConversion
   ```

3. **Choose an Option:**

   - Enter `1` to generate a conversion table.
   - Enter `2` to convert a single temperature value.

4. **Follow the On-Screen Instructions:**

   - For generating a table, you will be prompted to choose the starting and ending values.
   - For single-value conversion, you will need to enter the temperature value and the conversion type.

5. **View the Results:**

   The program will display the conversion results or the generated table.

6. **Repeat or Quit:**

   After viewing the results, you can choose to perform another conversion or exit the program.

7. **Exit the Program:**

   To exit the program, follow the on-screen instructions or simply close the terminal or command prompt.
